# SQL-Practice
This repository contains:
- SQL tasks for practicing basic and advanced queries

- Skills covered:
- Basic Queries (`SELECT`, `WHERE`, `ORDER BY`, `LIMIT`, `GROUP BY`, `HAVING`)
- Filtering & Conditions (`AND`, `OR`, `IN`, `BETWEEN`, `LIKE`)
- Data Transformations and Cleaning: String, Number, Date, Null Functions
- Case Statements `CASE WHEN`
- Aggregations (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- Joins (`INNER`, `LEFT`, `RIGHT`, `FULL`)
- Window functions (basic usage like `ROW_NUMBER`, `RANK`)
- Writing clean, readable, and maintainable SQL (CTE, Subqueries, Views, CTAS, Temp Tables )
